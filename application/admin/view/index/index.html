后台介绍

BearAdmin 是一个基于TP5+AdminLTE开发的简单后台管理系统，目前包含以下功能（模块）：

  • 基础功能
    • 用户管理
    • 角色管理
    • 菜单管理
    • 操作日志管理
    • 基于菜单的权限管理
  • 扩展功能
    • Ueditor集成
    • 邮件发送
    • 二维码生成
    • 阿里大于
    • Excel导入导出
    • 文件上传下载
  • 助手函数
    • 获取指定ip的地理位置信息及天气信息
    • 字符串的加密解密
  • 计划添加
    • 系统日志管理(待开发完善)
    • 七牛云存储(待添加)
    • 第三方登录(待添加)
    • 支付插件(待添加)
